Build a production-level AI healthcare frontend application named "Jeevansh AI".

Tech Stack (STRICTLY FOLLOW):

* React (Vite)
* Tailwind CSS
* Framer Motion (for animations)
* ShadCN UI (for components)

Project Goal:
Design a visually stunning, modern, and professional AI-powered medical diagnosis platform with a hospital-grade UI that feels like a billion-dollar healthcare startup product.

IMPORTANT:

* Frontend only (NO backend logic)
* Use mock data / dummy APIs
* Focus heavily on UI/UX, animations, and component reusability
* Clean and scalable folder structure

* Dark mode toggle
